# Notebooks

This directory contains the main notebooks used for Kaggle Playground Series S6E4: Irrigation Need Prediction.

## Notebook Overview

| Notebook | Description |
|---|---|
| `01_eda_and_feature_inspection.ipynb` | Exploratory data analysis and initial feature inspection. |
| `02_xgb_domain_features_single_model.ipynb` | XGBoost single-model pipeline with domain-inspired features, external data, target encoding, and log-odds bias tuning. |
| `03_xgb_reproduction_and_bias_tuning.ipynb` | Reproduction-style XGBoost experiments and bias adjustment variants. |
| `04_ensemble_and_submission_analysis.ipynb` | Ensemble experiments, OOF/Public LB comparison, prediction distribution checks, and final submission analysis. |

## Notes

The notebooks are intended to document the experiment process rather than provide a fully automated end-to-end pipeline.

Original competition data and generated prediction files are not included in this repository.
